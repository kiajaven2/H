الأصوات — Audio files
=====================

هنا يوضع ملفان صوتيان (كلاهما اختياري):
Two audio files go here (both optional):

1) الرسالة الصوتية 🎙️  —  the voice note  →  voice-note.mp3
   ------------------------------------------------------------------
   هذه هي الرسالة الصوتية التي تُشغَّل عند الضغط على "البالون الذهبي" في النهاية.
   This is the voice note that plays when the GOLDEN balloon is popped at the finale.
   • سجّلي رسالتك الصوتية واحفظيها باسم بالضبط:  voice-note.mp3  داخل هذا المجلد.
     Record your message and save it as exactly  voice-note.mp3  in this folder.
   • أو غيّري المسار في data/messages.js (حقل voice) ليطابق اسم ملفك.
     Or change the path in data/messages.js (the "voice" field) to match your filename.
   • صيغ مدعومة: mp3 (الأفضل)، m4a، ogg، wav.
     Supported formats: mp3 (best), m4a, ogg, wav.

2) موسيقى الخلفية 🎵 (اختياري) — background music (optional)  →  birthday.mp3
   ------------------------------------------------------------------
   تُشغَّل عبر زر 🎵 في الزاوية. ضعي ملفًا خاليًا من الحقوق باسم  birthday.mp3.
   Played via the 🎵 button in the corner. Drop a royalty-free track named  birthday.mp3.

ملاحظات / Notes:
  • إن لم تضعي أي ملف، كل شيء يعمل بدون أخطاء — البالون الذهبي يظهر لكن لن يُشغّل صوتًا.
    If no file is added, everything still works without errors — the golden balloon shows
    but simply won't play audio.
  • تشغيل الصوت تلقائيًا محظور، لذلك يبدأ الصوت فقط بعد ضغط المستخدم.
    Autoplay is blocked by browsers, so audio starts only after a user click.
